# CALY Project Website — V4

This version includes the requested design refinements after review.

## Included changes

- Reduced the hero headline size.
- The hero now uses a direct PNG file path:
  - `assets/images/hero-image.png`
- Removed the old hero placeholder composition and kept the hero visual as a clean image block.
- Refined the **Why CALY** section spacing.
- Made the **CALY solution** side the green highlighted card.
- Added zoom controls to the **Explore all interface screens** lightbox.
- Simplified **Meet the team** so each card shows only:
  - name
  - LinkedIn button
- Removed the repeated `Graduation Project Team` label.
- Removed the initials blocks like `KA`.
- Improved the layout and balance of the **Closed beta** section.

## Replace the hero PNG

Just replace this file with your final image:

```text
assets/images/hero-image.png
```

You can keep the same filename for the easiest workflow.

## APK location

Put the beta APK here:

```text
assets/downloads/CALY-Beta.apk
```

## LinkedIn links

Search for `placeholder-link` inside `index.html`, then replace each `href="#"` with the correct LinkedIn profile URL.
